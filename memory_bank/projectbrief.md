# Project Brief: Школьный Справочник

## Цель проекта

Разработка Telegram Mini App — интерактивного справочника формул и определений для школьников.

## Основные задачи

1. Создать удобный интерфейс для быстрого доступа к учебным материалам
2. Организовать структурированное хранение формул и определений по предметам
3. Обеспечить интеграцию с Telegram для нативного UX
4. Реализовать систему поиска и избранного

## Целевая аудитория

Школьники, которым нужен быстрый доступ к формулам и определениям через Telegram

## Ключевые метрики успеха

- Удобство навигации по предметам и темам
- Скорость доступа к нужной информации
- Качество отображения математических формул (LaTeX)
- Интеграция с Telegram (темы, haptic feedback)

## Ограничения

- Проект должен работать как Telegram Mini App
- Бесплатный tier Supabase и Vercel
- Поддержка светлой и темной темы
